61 Build
62 -----
63 
64 ### 1. Build CheMPS2 with CMake
65 
66 CheMPS2 can be built with CMake. The files
67 
68  ./CMakeLists.txt
69  ./CheMPS2/CMakeLists.txt
70  ./tests/CMakeLists.txt
71  ./PyCheMPS2/CMakeLists.txt
72 
73 provide a minimal compilation. Start in ```./``` and run:
74 
75  > mkdir build
76  > cd build
77 
78 CMake generates makefiles based on the user's specifications:
79 
80  > CXX=option1 cmake .. -DMKL=option2 -DBUILD_DOCUMENTATION=option3
81  -DCMAKE_INSTALL_PREFIX=option4
82 
83 Option1 is the c++ compiler; typically ```g++``` or ```icpc``` on Linux.
84 Option2 can be ```ON``` or ```OFF``` and is used to switch on the
85 intel math kernel library.
86 Option3 can be ```ON``` or ```OFF``` and is used to switch on doxygen
87 documentation.
88 Option4 is the prefix of the installation directory; typically
89 ```/usr``` on Linux. The CheMPS2 library is then installed in
90 ```prefix/lib``` and the headers in ```prefix/include/CheMPS2```.
91 If one or more of the required libraries are not found, please use the
92 command
93 
94  > CMAKE_INCLUDE_PATH=option5 CMAKE_LIBRARY_PATH=option6 CXX=option1
95  cmake .. -DMKL=option2 -DBUILD_DOCUMENTATION=option3
96  -DCMAKE_INSTALL_PREFIX=option4
97 
98 instead, where option5 and option6 are respectively the missing
99 colon-separated include and library paths, e.g.
100 
101  CMAKE_INCLUDE_PATH=/my_libs/lib1/include/:/my_libs/lib2/include/
102 
103 and
104 
105  CMAKE_LIBRARY_PATH=/my_libs/lib1/lib/:/my_libs/lib2/lib/
106 
107 To compile, run:
108 
109  > make
110 
111 To install, run:
112 
113  > make install
114 
115 For non-standard installation directories, please remember to append
116 the library path to ```LD_LIBRARY_PATH``` in your .bashrc.

196 Matrix elements from Psi4
197 -------------------------
198 
199 CheMPS2 has a Hamiltonian object which is able to read in matrix elements
200 from a plugin to [Psi4, Ab initio quantum chemistry](http://www.psicode.org),
201 which works on version psi4.0b5 and higher.
202 
203 To make use of this feature, build Psi4 with the plugin option, and then run:
204 
205  > psi4 --new-plugin mointegrals
206  > cd mointegrals
207 
208 Now, replace the file ```mointegrals.cc``` with either:
209 
210 1. [```mointegrals/mointegrals.cc_PRINT```](mointegrals/mointegrals.cc_PRINT)
211  to print the matrix elements as text. Examples of output generated with this
212  plugin can be found in [```tests/matrixelements```](tests/matrixelements)
213 
214 2. [```mointegrals/mointegrals.cc_SAVEHAM```](mointegrals/mointegrals.cc_SAVEHAM)
215  to store all unique matrix elements (remember that there is eightfold
216  permutation symmetry) in binary form with HDF5. See the Doxygen manual for
217  more information on CheMPS2::Hamiltonian.
218 
219 For case 2, the ```Makefile``` should be adjusted. Replace
220  ```$(PSILIBS)```
221 with
222  ```$(PSILIBS) -L${CheMPS2_BINARY_DIR}/CheMPS2/ -lCheMPS2```
223 and replace
224  ```$(CXXINCLUDE)```
225 with
226  ```$(CXXINCLUDE) -I${CheMPS2_SOURCE_DIR}/CheMPS2/include/```
227 
228 To compile the Psi4 plugin, run:
229 
230  > make

573 [```tests/test1.cpp.in```](tests/test1.cpp.in) contains several DMRG ground

582

584 calculation of the ^6A state of a linear Hubbard chain (forced c1 symmetry)
585 with 10 sites and open boundary conditions, containing 9 fermions (just below
586 half-filling).
587 
588 [```tests/test5.cpp.in```](tests/test5.cpp.in) contains an excited state DMRG
589 calculation in the ^1Ag symmetry sector of N2 (d2h symmetry) in the minimal
590 STO-3G basis set. The ground and two lowest excited states are determined.
591 
592 [```tests/test6.cpp.in```](tests/test6.cpp.in) contains a DMRG-SCF ground state
593 calculation of the ^1Ag state of O2 (d2h symmetry) in the CC-pVDZ basis set.
594 The two 1s core orbitals are kept frozen, and two Ag, B2g, B3g, B1u, B2u, and
595 B3u orbitals are chosen as active space. A significant speedup is obtained with
596 DIIS.

599 [```tests/matrixelements/O2_CCPVDZ.dat```](tests/matrixelements/O2_CCPVDZ.dat),
600 stores these matrix elements to disk, reads them back in from disk, and
601 compares the two versions.
602 
603 [```tests/test8.cpp.in```](tests/test8.cpp.in) contains a DMRG-SCF ground state
604 calculation of the ^1Ag state of N2 (d2h symmetry) in the CC-pVDZ basis set.
605 The two 1s core orbitals are kept frozen. The next two Ag and B1u orbitals
606 (sigma bonding and antibonding), as well as one B2g, B3g, B2u, and B3u orbital
607 (pi bonding and antibonding) are chosen as active space. A significant speedup
608 is obtained with DIIS. This test is smaller than test6, and is included for
609 debugging with valgrind.
610 
611 [```tests/test9.cpp.in```](tests/test9.cpp.in) is a copy of

613 space and which works with ordered localized orbitals instead of natural
614 orbitals. The localization occurs by means of Edmiston-Ruedenberg, and the
615 ordering based on the Fiedler vector with the exchange matrix as cost function.
