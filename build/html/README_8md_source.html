168 Matrix elements from Psi4
169 -------------------------
170 
171 CheMPS2 has a Hamiltonian object which is able to read in matrix elements
172 from a plugin to Psi4
173 [Psi4, Ab initio quantum chemistry](http://www.psicode.org),
174 which works on version psi4.0b5 and higher.
175 
176 To make use of this feature, build Psi4 with the plugin option, and then run:
177 
178  > psi4 --new-plugin mointegrals
179  > cd mointegrals
180 
181 Now, replace the file ```mointegrals.cc``` with either:
182 
183 1. ```./mointegrals/mointegrals.cc_PRINT``` to print the matrix elements as
184  text. Examples of output generated with this plugin can be found in
185  ```./tests/matrixelements```
186 
187 2. ```./mointegrals/mointegrals.cc_SAVEHAM``` to store all unique matrix
188  elements (remember that there is eightfold permutation symmetry) in
189  binary form with HDF5. See the Doxygen manual for more information on
190  CheMPS2::Hamiltonian.
191 
192 For case 2, the ```Makefile``` should be adjusted. Replace
193  ```$(PSILIBS)```
194 with
195  ```$(PSILIBS) -L${CheMPS2_BINARY_DIR}/CheMPS2/ -lCheMPS2```
196 and replace
197  ```$(CXXINCLUDE)```
198 with
199  ```$(CXXINCLUDE) -I${CheMPS2_SOURCE_DIR}/CheMPS2/include/```
200 
201 To compile the Psi4 plugin, run:
202 
203  > make

206 Build
207 -----
208 
209 ### 1. Build CheMPS2 with CMake
210 
211 CheMPS2 can be build with CMake. The files
212 
213  ./CMakeLists.txt
214  ./CheMPS2/CMakeLists.txt
215  ./tests/CMakeLists.txt
216 
217 provide a minimal compilation. Start in ```./``` and run:
218 
219  > mkdir build
220  > cd build
221 
222 CMake generates makefiles based on the user's specifications:
223 
224  > CXX=option1 cmake .. -DMKL=option2 -DBUILD_DOCUMENTATION=option3
225 
226 Option1 is the c++ compiler; typically ```g++``` or ```icpc``` on Linux.
227 Option2 can be ```ON``` or ```OFF``` and is used to switch on the
228 intel math kernel library.
229 Option3 can be ```ON``` or ```OFF``` and is used to switch on doxygen
230 documentation.
231 
232 To compile, run:
233 
234  > make
235 
236 
237 ### 2. Testing CheMPS2
238 
239 To test CheMPS2, start in ```./build```, and run:
240 
241  > cd tests/
242  > ./test1
243  > ./test2
244  > ./test3
245  > ./test4
246  > ./test5
247  > ./test6
248 
249 The tests should end with a line stating whether or not they succeeded.
250 They only require a very limited amount of memory (order 10-100 MB).
251 
252 ### 3. Doxygen documentation
253 
254 To build and view the Doxygen manual, the documentation flag should have
255 been on: ```-DBUILD_DOCUMENTATION=ON```. Start in ```./build``` and run:
256 
257  > make doc
258  > cd LaTeX-documents
259  > make
260  > evince refman.pdf &
261  > cd ../html
262  > firefox index.html &
