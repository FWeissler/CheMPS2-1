Detailed Description

CASSCF class.

Author
Sebastian Wouters sebas.nosp@m.tian.nosp@m.woute.nosp@m.rs@g.nosp@m.mail..nosp@m.com
Date
June 18, 2013

The CASSCF class provides an implementation of DMRGSCF [Zgid et al., J. Chem. Phys. 128, 144116 (2008)] with as DMRG solver the spin-adapted DMRG code CheMPS2. For the CASSCF update, the following implementations are included:

  • the quadratically convergent Newton-Raphson approach [Siegbahn et al., J. Chem. Phys. 74, 2384 (1981)]
  • the augmented Hessian Newton-Raphson approach [Lengsfield, J. Chem. Phys. 73, 382 (1980)]; which is ORCA's prefered choice
